## 🎯 Sobre o Projeto

Este projeto transforma mais de **13,9 milhões de registros dos Microdados do ENEM** em inteligência educacional para apoiar gestores públicos, pesquisadores, organizações sociais e formuladores de políticas públicas.

A solução combina análise territorial, indicadores educacionais e modelagem preditiva para identificar desigualdades regionais, monitorar desempenho educacional e projetar cenários futuros de evolução da educação brasileira.

O ecossistema foi desenvolvido a partir de dados oficiais do INEP e integra técnicas de Ciência de Dados, Estatística e Machine Learning para gerar diagnósticos e recomendações acionáveis.

---

## 🚀 Principais Resultados

* 📚 13,9 milhões de registros processados
* 🏙️ Cobertura nacional de municípios brasileiros
* 🏫 Mais de 40 mil escolas analisadas
* 📈 Projeções educacionais para 2026 e 2030
* 🤖 Modelos preditivos validados com dados históricos
* 🎯 Erro Absoluto Médio (MAE) de 32,7 pontos
* 🚨 Identificação automática de municípios críticos
* 🗺️ Visualização geoespacial e inteligência territorial

---

## 🧠 Problema de Negócio

Embora o ENEM represente uma das maiores bases educacionais da América Latina, seu volume e complexidade dificultam o uso estratégico para tomada de decisão.

Este projeto busca responder perguntas como:

* Onde estão os maiores focos de desigualdade educacional?
* Quais municípios apresentam maior risco de estagnação?
* Quais regiões possuem maior potencial de crescimento?
* Como políticas públicas podem impactar o desempenho educacional futuro?

## 📊 Soluções Desenvolvidas

### ENEM Vision

Plataforma de inteligência territorial focada em análise descritiva.

**Recursos:**

* Mapas interativos
* Rankings nacionais
* Séries temporais
* Comparativos regionais
* Indicadores educacionais e socioeconômicos
* Distribuições estatísticas e análise de outliers

---

### Predict Edu Impact

Plataforma de inteligência preditiva focada em projeções e simulação de cenários.

**Recursos:**

* Previsão de desempenho educacional
* Simulação de políticas públicas
* Classificação de municípios por criticidade
* Radar de vulnerabilidade
* Recomendações orientadas por dados
* Projeções até 2030

## 📈 Validação do Modelo

O modelo preditivo foi validado utilizando dados históricos entre 2015 e 2023.

Métrica principal:

* MAE (Mean Absolute Error): 32,7 pontos

A validação permitiu comparar valores previstos e observados para centenas de municípios brasileiros, garantindo maior confiabilidade para projeções futuras.
