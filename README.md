## 🎯 Sobre o Projeto

Este projeto transforma mais de **13,9 milhões de registros dos Microdados do ENEM** em inteligência educacional para apoiar gestores públicos, pesquisadores, organizações sociais e formuladores de políticas públicas.

A solução combina análise territorial, indicadores educacionais e modelagem preditiva para identificar desigualdades regionais, monitorar desempenho educacional e projetar cenários futuros da educação brasileira.

---

## 🎯 Objetivos

* Transformar microdados educacionais em inteligência acionável
* Identificar desigualdades regionais no desempenho escolar
* Apoiar a formulação de políticas públicas baseadas em evidências
* Desenvolver modelos preditivos para projeções educacionais
* Disponibilizar visualizações acessíveis para gestores e pesquisadores

---

## 🧠 Problema de Negócio

Embora o ENEM represente uma das maiores bases educacionais da América Latina, seu volume e complexidade dificultam o uso estratégico para tomada de decisão.

O projeto busca responder questões como:

* Onde estão os maiores focos de desigualdade educacional?
* Quais municípios apresentam maior risco de estagnação?
* Quais regiões possuem maior potencial de crescimento?
* Como políticas públicas podem impactar os resultados futuros?

---

* 📚 13,9 milhões de registros processados

## 📈 Validação do Modelo

Validação realizada com dados históricos entre 2015 e 2023.

### Métrica Principal

* MAE (Mean Absolute Error): 32,7 pontos

O processo permitiu comparar valores previstos e observados para centenas de municípios brasileiros, aumentando a confiabilidade das projeções futuras.
